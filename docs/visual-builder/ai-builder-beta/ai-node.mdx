# AI node

**AI node** generates a JavaScript node from a request description. Provide a cURL command or describe what data to send and where — the node is created with all the necessary fields.

## How it works

The AI node interface works the same way as the [JavaScript node](/visual-builder/code-tools/javascript) chat: you enter a request description, and the output is a ready-configured JavaScript node. The assistant determines the method, URL, headers, and request body automatically.

## How to use

1. Click the **AI node** button in the bottom menu of the builder — an input window will open.
2. Enter a cURL command or describe the request: the service, endpoint, and what data needs to be sent.
3. The assistant creates a JavaScript node with all the necessary fields — method, URL, parameters, and request body.
4. Review the generated configuration and adjust manually if needed.
