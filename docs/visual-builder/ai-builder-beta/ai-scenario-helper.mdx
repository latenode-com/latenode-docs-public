## Scenario-wide assistant

The **AI scenario helper** in **Latenode Workflow Builder** helps you create, refine, and debug workflows using natural language.

Instead of manually picking nodes and wiring everything step-by-step, you can describe your goal and let the helper propose a workflow structure, connect nodes, and prefill configuration where possible. This can significantly reduce the time it takes to get a working scenario – especially for first drafts and troubleshooting.

## Working with the helper

1. **Describe what you want to automate**: start from an example prompt or type your requirements in plain language (what triggers the workflow, what data should be used, and what the result should be).
2. **Monitor the build process**: the helper shows progress as it iterates through planning, selecting nodes, connecting them, and setting parameters.
3. **Review and refine**: verify credentials, required inputs, and any assumptions; then ask the helper to adjust the workflow (for example, add error handling, logging, or additional branches).
